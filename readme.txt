== Description ==
A complete statistics of your website. Site Statistics - Internal Web Stats Plugin gives you a glimpse of the statistics of posts, pages and comments you have in your website. Also, it shows the server information too.

== Installation ==
1. Unzip the downloaded .zip file.
2. Upload the plugin folder to the wp-content/plugins/ directory of your WordPress site.
3. Go to the 'Plugins' menu in WordPress and activate the Site Statistics - Internal Web Stats.
4. Visit Dashboard for the added wiget boxes - Server Information, Blog Posts & Pages Informations
